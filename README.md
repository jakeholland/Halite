# Halite
![Halite Icon](/Halite/Assets.xcassets/AppIcon.appiconset/Icon-App-76x76@2x.png?raw=true "Halie Icon")

Halite is a free winter road conditions map iOS app, written in [Swift](https://developer.apple.com/swift).

## Supported States
- North Dakota
- South Dakota
- Nebraska
- Kansas
- Missouri
- Illinois
- Wisconsin
- Minnesota
- Iowa
